Description des méthodes

Méthode : getListOfItems
  • Classe : ItemController
  • Package : Controllers
  • Description : Retourne une liste des articles.
  • URL : /resume/xml
  • Méthode HTTP : GET
  • Paramètres d'entrée : -
  • Retour Succès : Retourne une liste des articles.
  • Type de retour : XML
  • Retour Erreur : -
Méthode : getItemByGuid
  • Classe : ItemController
  • Package : Controllers
  • Description : Retourne un article.
  • URL : /resume/xml/{guid}
  • Méthode HTTP : GET
  • Paramètres d'entrée : guid de type String
  • Retour Succès : Retourne un article.
  • Type de retour : XML
  • Retour Erreur : Retourne une exception si guid null ou guid n'existe pas
Méthode : createItem
  • Classe : ItemController
  • Package : Controllers
  • Description : Creation d'un article.
  • URL : /insert
  • Méthode HTTP : POST
  • Paramètres d'entrée : Un objet Item.
  • Retour Succès : Retourne l'Item créé.
  • Type de retour : XML
  • Retour Erreur : Retourne BAD_REQUEST si les données envoyées sont invalide.
Méthode : deleteItem
  • Classe : ItemController
  • Package : Controllers
  • Description : Suppression d'un article.
  • URL : /delete/{guid}
  • Méthode HTTP : DELETE
  • Paramètres d'entrée : Le guid de l'article.
  • Retour Succès : Retourne l'article supprimé.
  • Type de retour : XML
  • Retour Erreur : Retourne RunTimeException si le guid n'existe pas dans la BD.
Méthode : findItemList
  • Classe : ItemController
  • Package : Controllers
  • Description : Retourne liste des articles.
  • URL : /resume/html
  • Méthode HTTP : GET
  • Paramètres d'entrée : Le guid de l'article.
  • Retour Succès : Retourne liste des articles format HTML.
  • Type de retour : HTML
  • Retour Erreur : -
Méthode : findItemByGuid
  • Classe : ItemController
  • Package : Controllers
  • Description : Retourne liste des articles.
  • URL : /resume/html/{guid}
  • Méthode HTTP : POST
  • Paramètres d'entrée : Le guid de l'article.
  • Retour Succès : Retourne un article format HTML.
  • Type de retour : HTML
  • Retour Erreur : Retourne RunTimeException si le guid n'existe pas dans la BD.
